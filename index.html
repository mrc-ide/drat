mrc-ide drat repository

MRC IDE packages that are not yet (and in some cases ever) going on CRAN

Packages

  • buildr (v 0.0.3, ): buildr
  • cinterpolate (v 1.0.0, ): Interoplation From C
  • context (v 0.1.4, ): Contexts for evaluating R expressions
  • dde (v 0.0.5, ): Solve Delay Differential Equations
  • didehpc (v 0.2.0, ): DIDE HPC Support
  • odin (v 0.0.3, ): Ode Generation and Integration
  • provisionr (v 0.1.10, ): Provision a Library
  • queuer (v 0.1.2, ): Queue Tasks
  • rcmdshlib (v 0.0.2, ): Build Shared Libraries From R
  • rrq (v 0.1.2, ): Simple Redis Queue
  • syncr (v 0.0.3, ): Synchronise Directories

If drat is not installed, first install from CRAN with

install.packages("drat")

To install packages, run

drat:::add("mrc-ide")

after which packages can be installed as usual, for example:

install.packages("provisionr")
